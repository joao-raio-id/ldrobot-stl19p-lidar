# ldrobot-lidar 

This is a ROS2 package based on the official package from LDRobot:
```bash
git@github.com:ldrobotSensorTeam/ldlidar_ros2.git
```

And the one made by Myzhar with documentation available here[www.waveshare.com/wiki/DTOF_LIDAR_LD19]:
```bash
git@github.com:Myzhar/ldrobot-lidar-ros2.git
```

## Composition
From LDRobot comes simple nodes that use the official SDK.
From Myzhar comes lifecycle nodes that use improved and modified libraries based on the official SDK
